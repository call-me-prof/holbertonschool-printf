_printf

Custom implementation of the printf function in C.

⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻

Description

This project is a recreation of the standard printf function from the C library.

The goal is to understand how formatted output works internally by implementing our own version called _printf, handling different format specifiers and arguments.

⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻

How It Works

_printf reads a format string and prints output to the standard output (stdout).

* It scans the string character by character
* When it finds %, it checks the specifier
* Calls the correct function to handle it
* Prints the result

Requirements

* Allowed editors: vi, vim, emacs
* Compiled using: gcc -Wall -Werror -Wextra -pedantic -std=gnu89
* No global variables allowed
* Maximum 5 functions per file
* Code follows Betty style

⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻

Supported Format Specifiers

* %c → character
* %s → string
* %d → integer
* %i → integer
* %% → percent sign

⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻

Project Files

* main.h → header file (prototypes)
* _printf.c → main function
* Other .c files → handlers and helper functions

⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻

Compilation & Execution

Compile the project: gcc -Wall -Werror -Wextra -pedantic *.c -o printf
Run the program: ./printf

⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻⸻

Example

#include "main.h"

int main(void)
{
    _printf("Hello %s!\n", "World");
    _printf("Number: %d\n", 100);
    return (0);
}
